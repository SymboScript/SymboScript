# SymboScript Lexer

## Roadmap

- [x] Cli
- [x] Error handling
- [ ] Full language support

## Full language support

- [x] Standard operators
- [x] Functions
- [x] Comments
- [x] Strings
- [x] Numbers
- [x] Variables
- [x] Identifiers
- [x] Keywords
- [ ] Other?

## Cli install

```sh
cargo install symboscript-lexer
```

## Cli usage

```sh
symboscript-lexer -h
```

## Run examples

```sh
symboscript-lexer ../examples/lexer_test.syms
```
